## Idea 1: "Obsidian Command Center" — Tactical Precision Design

**Design Movement:** Military-grade command interface meets Apple spatial computing.
**Core Principles:** Information density with breathing room; luminance as hierarchy; kinetic stillness; zero visual waste.
**Color Philosophy:** Deep obsidian (#060C14) with cold cyan (#5BB6C9 to #9CF3FF) for system elements, warm amber (#F5A623 to #FFD700) for value/ROI, neutral silver (#C8D1DA) for body.

**Signature Elements:** Pulsing dot indicators, luminous divider lines, numbered module cards with parallax depth.
**Interaction Philosophy:** Hover reveals depth. Scroll triggers sequential reveals. Nothing moves unless user causes it.
**Animation:** Intersection Observer fade-up + stagger. Counter animations for stats. Subtle pulse on live indicators.
**Typography System:** Inter Tight for headlines (-0.06em, semibold, 5rem+), Inter for body (1.08rem), uppercase mono labels.

## Selected Approach: Idea 3 — "Spatial Glass" (Apple-Inspired Depth Design)

This approach best serves CasePort's goals because:

1. Apple = premium perception. The user explicitly said "CasePort is the Apple of case acquisition."
2. Inhale/exhale rhythm is built into the layout paradigm.
3. The restraint signals confidence — a new company needs "we don't need to try hard" energy.
4. The golden ROI calculator creates the single interactive spectacle that drives conversion.
5. Cinematic scroll experience makes the page feel like a product reveal, not a marketing page.
