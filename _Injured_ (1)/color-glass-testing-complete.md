- **Standard Glass Panel**: 12px blur, 75% opacity, consistent borders and shadows
- **Glass Variants**:

  - `.glass-panel-gold` — Gold tint, value emphasis
  - `.glass-panel-red` — Red tint, urgency
  - `.glass-panel-deep` — Layered depth effect
- **Hover Effects**: All glass panels have smooth transitions with enhanced blur and border color changes
- **Consistency**: All glass effects use same blur value (12px) and opacity (75%) for unified aesthetic
