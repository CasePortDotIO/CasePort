# CasePort Homepage Design Benchmark Analysis

## Key Design Observations

### Typography & Hierarchy
- **Headline Style**: Bold, large, multi-line headlines with color accents (cyan highlights on key words)
- **Example**: "Stop Buying Leads. Start Controlling Case Flow." (with "Start Controlling" in cyan)
- **Font Weight**: Heavy, bold fonts for impact
- **Color Strategy**: White primary text with cyan accent highlights on key concepts

### Color Palette
- **Primary**: Dark background (slate-900/black)
- **Accent**: Cyan (#06b6d4 or similar) - used strategically on key words and CTAs
- **Accent 2**: Amber/Gold (#fbbf24) - used for warnings/important notes
- **Text**: White and light grays

### Visual Elements
- **Background**: Dark, possibly with subtle texture or grid pattern
- **Buttons**: Cyan background with dark text (Request Private Access)
- **Icons**: Minimal, clean, used sparingly
- **Dividers**: Thin lines, subtle

### Layout & Spacing
- **Hero Section**: Large, commanding presence
- **Case Flow Engine**: Visual diagram showing 6 steps (Search Intent → Demand Capture → Qualification → Routing → Recovery → Retained Value)
- **Grid System**: Appears to use a clean grid with generous spacing
- **Sections**: Clear visual separation between sections

### Micro-Interactions & Motion
- **Buttons**: Likely have hover effects with scale/color changes
- **Interactive Elements**: Smooth transitions
- **Animations**: Subtle, purposeful

### Key Sections Visible
1. **Hero** - Dark, bold, with cyan accents
2. **Case Flow Engine** - Visual diagram with 6 numbered steps
3. **Market Intelligence** - Large statistics ($35B, 3M+, $855, $28K-500K)
4. **The Problem** - "Most Lead Gen Is Broken by Design"
5. **Comparison** - "Without Discipline" vs "With CasePort"

### Design Philosophy
- **Tone**: Premium, institutional, confident
- **Approach**: Data-driven, visual storytelling
- **Aesthetic**: Dark mode, high contrast, strategic color accents
- **Feel**: Professional, sophisticated, authoritative

## Key Differences from Current Claimant Page
- Current page: Light background (white)
- Homepage: Dark background (slate-900/black)
- Current page: Multiple colors (cyan, blue, purple, amber, rose, red)
- Homepage: Primarily dark with cyan and amber accents
- Current page: More colorful, varied
- Homepage: More restrained, strategic color use

## Design Gaps in Current Claimant Page
1. **Background**: Should consider dark mode option or match homepage aesthetic more closely
2. **Color Restraint**: Too many accent colors - should be more strategic (cyan + amber)
3. **Typography Accents**: Should use color highlights on key words like homepage does
4. **Visual Diagrams**: Homepage uses visual flow diagrams effectively
5. **Statistics/Data Presentation**: Homepage presents data in large, bold format
