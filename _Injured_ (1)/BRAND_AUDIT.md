# CasePort Brand Audit — Claimant Page vs. Law Firm Page

## Critical Gaps & Missing Elements

### 1. **VISUAL DESIGN GAPS**

#### Missing: Neon Green Accent Color (#00FF00 / Lime)
- **Law Firm Page**: Uses bright neon green (#00FF00) for:
  - Dashed borders on cards
  - Accent highlights
  - Visual pop elements
  - Danger/warning indicators
- **Claimant Page**: Uses only cyan (#22D3EE) and gold (#F59E0B)
- **Action**: Add neon green accents for visual consistency and institutional credibility

#### Missing: Neon Magenta/Purple Gradient
- **Law Firm Page**: Uses vibrant magenta-to-purple gradient on buttons and CTAs
- **Claimant Page**: Uses only cyan-to-cyan gradient
- **Action**: Implement magenta-to-purple gradient on secondary CTAs for visual hierarchy

#### Missing: Countdown Timer with Visual Progress Bar
- **Law Firm Page**: Features "LIMITED AVAILABILITY" countdown (01 days, 23 hours, 59 minutes, 33 seconds) with:
  - Cyan-to-magenta gradient progress bar
  - Neon green accent on time units
  - Urgency-driven design
- **Claimant Page**: No countdown timer
- **Action**: Add countdown timer to hero section with matching gradient

#### Missing: Dashed Border Design Pattern
- **Law Firm Page**: Uses dashed cyan borders on cards and sections for visual distinction
- **Claimant Page**: Uses solid borders only
- **Action**: Add dashed borders to key sections (CTAs, cards, highlights)
