# Reference Page Visual System Audit

## Design System Analysis

### Color Palette
- **Primary Background**: Very dark navy/black (#0A0E17 or similar)
- **Neon Cyan**: #00D9FF (used for accents, CTAs, highlights)
- **Neon Green**: #00FF00 (used for badges, dashed borders, status indicators)
- **Neon Red**: #FF0000 (used for urgency, "LAST SLOT", alerts)
- **Gold/Yellow**: #FFB800 (used for accents, system status)
- **Text**: White/light gray on dark background

### Typography
- **Display Font**: Bold, modern serif (possibly Instrument Serif or similar)
- **Body Font**: Clean sans-serif (Geist Sans or similar)
- **Labels**: Uppercase, tracking-widest, small font size
- **Hierarchy**: Clear size differentiation (H1 large, H2 medium, body small)

### Visual Elements
1. **Dashed Borders**: Neon green dashed borders on CTAs and key elements
2. **Status Badges**: Color-coded (RED=Limited, CYAN=Open, YELLOW=In Review, GRAY=Capped)
3. **Market Cards**: Dark glass morphism panels with border, showing:
   - Market name and status
   - Case count
   - Settlement range
   - Slots available
   - MII Score
4. **System Status**: Top bar showing "46 Markets Active", "5 Capped", etc.
5. **Testimonials**: Carousel with navigation dots
6. **Grid Layout**: Multi-column market grid with search/filter

### Key Design Patterns
- Dark immersive background
- Neon color accents (cyan, green, red, gold)
- Glass morphism panels
- Dashed borders for emphasis
- Uppercase labels for hierarchy
- Real-time data display (market status, counts)
- Color-coded status system
- Scarcity signals ("LAST SLOT", "CAPPED", "WAITLIST")
- Countdown timers ("CLOSES IN 47D")

### CTA Design
- Neon cyan buttons with dashed green borders
- Bold, action-oriented copy
- High contrast for visibility
- Hover states with glow effects

## Claimant Page Alignment Needed

The current claimant page is **white background**, but the reference page is **dark navy with neon accents**. 

**Options**:
1. **Full alignment**: Convert claimant page to dark navy + neon accents (matches brand)
2. **Hybrid approach**: Keep white background but add neon cyan/green accents and dashed borders (maintains light aesthetic while adding brand DNA)
3. **Premium white**: Keep white but add more sophisticated neon accent usage and glass morphism

**Recommendation**: Hybrid approach maintains the white background you love while adding CasePort's premium neon aesthetic.
