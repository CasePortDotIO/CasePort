# CasePort Brand System — Exact Specifications

## Color Palette

**Primary Colors:**
- **Dark Navy Background**: #0A0E17 (very dark, almost black)
- **Neon Cyan/Turquoise**: #00D9FF (bright, electric)
- **Neon Green**: #00FF00 (bright, electric)
- **Neon Red/Magenta**: #FF0000 (bright, electric, used for scarcity)
- **Gold/Yellow**: #FFB800 (warm, used for metrics)

**Text Colors:**
- **White**: #FFFFFF (primary text on dark)
- **Light Gray**: #CCCCCC (secondary text)
- **Dark Gray**: #666666 (tertiary text)

## Typography

- **Display Font**: Bold, heavy weight (appears to be sans-serif, possibly custom)
- **Body Font**: Clean sans-serif (Geist or similar)
- **Accent Color in Copy**: Cyan (#00D9FF) used inline for emphasis (e.g., "$15,000–$50,000" in cyan)

## Design Patterns

1. **Dashed Borders**: Neon green dashed borders around key elements (CTAs, cards)
2. **Color-Coded Status System**:
   - GREEN: Available, open, positive
   - CYAN: Active, current, information
   - RED/MAGENTA: Limited, scarcity, urgent
   - YELLOW/GOLD: Metrics, value, ROI

3. **Uppercase Labels**: Section labels in uppercase with letter-spacing (e.g., "CASE ACQUISITION SYSTEM", "OPERATIONAL")

4. **Neon Glow Effects**: Subtle glow on neon elements (cyan, green, red)

5. **Dark Immersion**: Dark navy background creates premium, exclusive feeling

6. **Dashed Border Treatment**: Signature element on CTAs and key sections

## Key Visual Elements

- **Neon Green Dashed Borders** on buttons and cards
- **Cyan Inline Text** for emphasis in copy
- **Red/Magenta Badges** for scarcity ("2 slots available")
- **Gold/Yellow Metrics** for ROI and value
- **Uppercase Labels** for hierarchy and urgency

## Tone

- **Institutional, Premium, Tech-Forward**
- **Urgent but Professional**
- **Data-Driven, Infrastructure-Focused**
- **Exclusive, Limited Availability**

## Application to Claimant Page

The claimant page should use:
- Dark navy background (#0A0E17)
- Neon cyan accents (#00D9FF)
- Neon green dashed borders on key elements
- Neon red for urgency/scarcity signals
- Gold for metrics and value
- Uppercase section labels
- Same premium, tech-forward aesthetic
- BUT: White background concept (user preference)

**CONFLICT**: User wants white background but brand system is dark navy. Need to find hybrid approach that maintains brand while respecting white background preference.
